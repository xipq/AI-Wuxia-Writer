# AI-Wuxia-Writer
Wuxia style novel generation using T5-PEGASUS model. 中文武侠小说续写

- 基于NLP的武侠小说写作模型
- 模型采用T5-PEGASUS（基于`hugging_face`，很容易更换为其他模型）
- 训练数据为金庸先生的著作（见`dev_data`和`train_data`）
